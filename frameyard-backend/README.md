# FrameYard Backend

## Project Overview

FrameYard is a custom frame ecommerce platform that allows users to:

* Upload their images
* Select frame styles and sizes
* Preview customizations
* Place orders online

This repository contains the backend services for the FrameYard platform.

---

### Backend
